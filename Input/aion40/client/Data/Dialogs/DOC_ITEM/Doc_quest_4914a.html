생체 개조 진행 상황

1. 아인종

- 브라우니: 세뇌 완료. 크랄에 비해 지능이 낮아 드라겔을 거부하지 않고 일부분이나마 받아들인 것으로 보임.

- 크랄: 드라겔 주입 실패. 포악하고 순종적이지 않은 성격 탓인지 정신감응 제어기의 명령을 받아들이지 못하고 발광 상태를 보이다 전멸함.

- 펑기: 애초 드라겔 주입 대상은 아니었지만 일부 나가의 실수로 드라겔을 투입한 개체가 만들어짐. 생태를 관찰하면서 활용 방안을 찾는 중.

2. 무생물

- 루크하: 숲과 벌목장에 말라 죽은 나무 둥치에 드라즈마를 투입하여 생성. 이용 가치가 높은 것으로 판단돼 재배지를 조성하여 양산 체계를 확립하려 함.

- 마라바타: 마을 앞에 있는 호수에 드라즈마의 핵을 투입하여 생성. 제어 장치를 이용하여 호수의 기운을 마라바타에 전달하고 있음.

3. 정신감응 제어장치

트몰리아 폐광에 설치. 아인종과 무생물, 각종 하위 제어장치를 효율적으로 제어하고 있음.

4. 기타

- 계속적으로 드라겔과 드라즈마를 투입하려면 드라나 생산이 필수적임. 현재 드라나 수확장에서 순조롭게 생산이 진행되고 있음.

- 차원의 문 동력 공급장치에 막대한 양의 수르카나가 소모되고 있음. 비상용 수르카나를 축적할 필요 있음.
